Blog #3: Week 7

In this week of my blog, I have learned many new tricks for CSS to make my websites look more eye-popping to the readers. This video in particular, showed me how to use an image as a background for a container. All you need to do is add the image file inside a ‘url()’ command and instantiate the background with it. Not only that, you could also give the background a color overlay that, in some cases, makes the image less stressful on the eyes and blend in well with the website. To achieve this, make your container’s position relative and make a div element, with a background color of your choosing, inside of the container fit the entire container then make the element’s opacity equal less than one so you can actually see other items inside that container. Lastly from the video, a very useful trick was making an element’s z-index equal one. This allows the element to be in front of everything so the color overlay, or anything else, will not hide/interfere with it.


Other neat things I have learned were from discussion boards or the w3s website. One thing that was awesome was being allowed to make your own scrollbar. You can change the color of the scroll bar by using this in your CSS file 

*.class*::-webkit-scrollbar { 
    width: 11px;
    height: 11px;
    background-color: lightgrey;
}

*.class*::-webkit-scrollbar-thumb { 
    background: blue;
}

Another amazing trick, from w3schools , was being able to store a html file into an element for your current file so it would be able to show itself in your screen.
