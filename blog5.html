Blog #5: Week 9

This week on my blog, I learned a very cool trick with HTML’s background. You could set the background to multiple colors intersecting. All that needs to be done is setting the “background-image” of an element to linear-gradient() like so: 

#example-div { 
    width: 100px;
    height: 100px;
    background-image: linear-gradient(to bottom right, blue, yellow);
}

example-div
If you want to add more than two colors, you can do so by just inserting colors after the position (in this case it is ‘to bottom right’). This will make my website more eye-popping, just like this other effect that is known as fade-in. This allows a HTML page to just fade in which I think will look so cool.


Another very cool thing I learned was a loading element. As the name suggests, it is an element that appears to be a circular loading object. This will be useful for my website’s opening because I can use this loading element to show up instead of my actual elements loading up and looking bad while doing so. I saw that there is also a progress bar that is used by the <progress> or <meter> tag, which could also be used as loading objects. The only problem is they stay at one position, so to increase the progress bar you need to update it using JavaScript, which is a hassle. Although, this seems useful for when I would like users to download a file so while waiting, they get a progress bar to know how long it will take.
