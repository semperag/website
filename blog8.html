This week was for me to catch up on things in my CS 347 class. I, am pretty sure that I, understand what endpoints mean now. They could just be characterized as a “URL of a server or service.” You set an endpoint for your code so that your are able to use helper methods of a particular server in order to work on or around that server much easier or faster. For example, Google Cloud gives you an API endpoint so that you are able to click on any button or search for anything by just running your code, obviously your code has to giving an order to your machine to access these accessories. With finding out what are endpoints, I also found out that they must be secured. It is best to use HTTPS websites to not risk using a sketchy one. If I am not wrong, my professor told me to limit the calls of a client so the system will not be overflowing by a coding error or bots.
